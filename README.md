## Running
Create a `.env` file in the root of the project with the content copied of `env-sample`. 

	yarn install
	netlify dev

Visit http://localhost:8888/

## Running the CMS

	npx netlify-cms-proxy-server
	netlify dev

Visit http://localhost:8888/admin/

## Building

	gatsby build
